Simple Mongo DB Schema Visualizer in HTML

It's a Mongo DB HTML Horizontal Tree View visualizer using variety.js [https://github.com/variety/variety] and pymongo.

Features:
	Show Variety info in a friendly user interface
	Show the 30 most common values on each field with %

How to use:
	Edit the setup.py file with your Database Info and run it! [ONCE]
	After setup you only need to run DATABASE_TREE_VIEW.py
	Done! 
	Will create the file DB_VIEW_HTML.html which have all that you need in one HTML file!

It's in alpha state
